dec_to_oct
==========

Converts a decimal value to a string of octal digits.

    oct = dec_to_oct(123);          //  oct == "173"
    oct = dec_to_oct(456);          //  oct == "710"
    oct = dec_to_oct(789, 6);       //  oct == "001425"
    oct = dec_to_oct(-43, 6);       //  oct == "777725"
