dec_to_hex
==========

Converts a decimal value to a string of hexadecimal digits.

    hex = dec_to_hex(123);          //  hex == "7B"
    hex = dec_to_hex(456);          //  hex == "1C8"
    hex = dec_to_hex(789, 4);       //  hex == "0315"
    hex = dec_to_hex(-43, 4);       //  hex == "FFD5"
