ds_map_clone
============

Clones a map simply to a new copy and returns the created map.

    map = ds_map_create();                  //      resulting map clone:
    map[? "Hello"] = "World";               //      {
    map[? "Goodbye"] = "Cruel World";       //          Hello: "World"
                                            //          Goodbye: "Cruel World"
    clone = ds_map_clone(map);              //      }
